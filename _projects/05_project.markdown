[PanPhon](https://github.com/dmort27/panphon) is a database mapping betweem (phonological) articulatory features and IPA segments as well as a library for manipulating and querying feature-based and segmental representations.

If you use PanPhon in research, please cite the [following paper](https://www.aclweb.org/anthology/C/C16/C16-1328.pdf):

David R. Mortensen, Patrick Littell, Akash Bharadwaj, Kartik Goyal, Chris Dyer, Lori Levin (2016).
